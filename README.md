rCrimemap
=========

This is the next generation of CrimeMap!


### Prerequisites

```
require(devtools)
install.packages("base64enc", "ggmap", "rjson", "dplyr")
install_github('ramnathv/rCharts@dev')
install_github('ramnathv/rMaps')
```

### Install rCrimemap

```
require(devtools)
install_github('woobe/rCrimemap')
```
